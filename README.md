# cn-m-hub

## 项目简介

这是一个用于归档和发布独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为静态 HTML 文件的集中存放与版本管理之用。每个 HTML 页面均保持独立，不依赖外部服务或数据库。

## 目录说明

```
/
├── index.html          # 仓库入口页（可选）
├── pages/              # 存放所有独立 HTML 页面的目录
│   ├── page-001.html
│   ├── page-002.html
│   └── ...
└── assets/             # 页面共用的资源文件（CSS、JS、图片等）
    ├── css/
    ├── js/
    └── images/
```

- `pages/` 目录下存放独立的 HTML 文件，每个文件为一个完整的页面。
- `assets/` 目录存放页面可能用到的公共资源，按类型分文件夹管理。

## 页面归档说明

- 所有 HTML 页面均为独立文件，可直接在浏览器中打开使用。
- 页面内容遵循通用 HTML 规范，不依赖特定后端环境。
- 页面命名建议使用简洁、有意义的英文或数字组合，避免特殊字符。
- 如需引用公共资源，请使用相对于仓库根目录的路径。

## 维护说明

- 本仓库由维护者定期更新，添加新的页面或修正已有内容。
- 欢迎通过 Issue 提交页面建议或问题反馈。
- 提交新页面时，请确保文件完整性，并放置在 `pages/` 目录下。
- 请勿在页面中包含恶意代码、广告弹窗或违反法律法规的内容。
